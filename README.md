# agl_static_site_template

## Setup

Match local node.js version with project version

```sh
nvm use
```

Install node_modules

```sh
npm install
or npm i -g pnpm && pnpm i
```

## Getting Started

Start cording

```sh
npm run dev
or pnpm run dev
```

> http:localhost:3000

Build finished data

```sh
npm run build
or pnpm run build
```
